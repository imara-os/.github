> Imara /ɪˈmɑːrə/ — Swahili for “strong” or “resilient.”

Imara is the desktop everyone has been waiting for: modern, easy to use, secure, and private. It builds on reliable and battle-tested open-source technologies and makes them work together beautifully.

## Before You Continue!

Please note that Imara is currently nothing more than a hobby project for [me](https://github.com/kurimanju-dev/) where I test different architectural approaches to see how each behaves and would suit Imara best. It is not available for download, yet. If you were looking for an OS that's ready to use and battle-tested, I suggest [Fedora Silverblue](https://fedoraproject.org/atomic-desktops/silverblue/).

## Additional technical information (not complete)

- Base: Immutable [Void Linux](https://voidlinux.org/) (glibc)
- Package Manager: spx (safe xbps/homebrew/flatpak-wrapper)
- Desktop Environment: imUIX (based on [COSMIC](https://system76.com/cosmic))
- Application Toolkit: libimUI (based on [libcosmic](https://github.com/pop-os/libcosmic))
